## 🎯 核心概念

### CSS 变量
使用 `--template-*` 前缀的 CSS 变量系统，与 `--color-*` 和 `--size-*` 保持一致。

### 主题管理
支持 Light/Dark/Auto 三种模式，运行时切换，跟随系统主题。

### 生态集成
与 @ldesign/color 和 @ldesign/size 深度集成，统一的设计语言。
