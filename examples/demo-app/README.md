# 🎨 LDesign Template 演示应用

这是一个完整的 Vite + Vue3 演示应用，展示了 `@ldesign/template` 的所有核心功能和高级特性。

## ✨ 功能特性

### 🚀 基础演示
- **模板渲染** - 实时渲染不同设备的登录模板
- **设备检测** - 自动检测并适配桌面、平板、手机设备
- **模板切换** - 动态切换不同风格的模板
- **属性配置** - 实时修改模板属性并查看效果
- **事件处理** - 完整的事件监听和处理演示

### 🔥 高级功能
- **缓存管理** - 模板缓存的增删改查操作
- **性能监控** - 实时监控加载时间、渲染时间等指标
- **主题定制** - 可视化主题编辑器，支持颜色、字体等自定义
- **错误处理** - 错误模拟和降级策略演示

### 👨‍💻 开发工具
- **代码编辑器** - 在线编辑模板配置、组件和样式
- **模板生成器** - 可视化生成自定义模板
- **属性编辑器** - 动态添加和编辑模板属性
- **事件编辑器** - 配置模板事件和参数
- **模板验证** - 验证模板配置的正确性

### 📊 性能监控
- **实时指标** - CPU、内存、网络等实时监控
- **性能分析** - 瓶颈分析和优化建议
- **趋势图表** - 性能数据的可视化展示
- **报告生成** - 导出详细的性能报告

## 🚀 快速开始

### 安装依赖

```bash
# 进入演示应用目录
cd packages/template/examples/demo-app

# 安装依赖
pnpm install
```

### 启动开发服务器

```bash
# 启动开发服务器
pnpm dev

# 应用将在 http://localhost:3000 启动
```

### 构建生产版本

```bash
# 构建生产版本
pnpm build

# 预览生产版本
pnpm preview
```

## 🎯 使用指南

### 基础演示

1. **模板渲染区域** - 查看当前选中模板的渲染效果
2. **设备模拟器** - 点击不同设备按钮模拟设备切换
3. **模板选择器** - 选择不同的模板进行切换
4. **属性配置** - 实时修改模板属性
5. **事件日志** - 查看模板触发的事件

### 高级功能

1. **缓存管理** - 清空缓存、预加载模板、查看缓存统计
2. **性能监控** - 查看加载时间、渲染时间等性能指标
3. **主题定制** - 使用颜色选择器和滑块自定义主题
4. **错误处理** - 模拟各种错误情况，测试降级策略

### 开发工具

1. **代码编辑器** - 查看和编辑模板的配置、组件、样式文件
2. **模板生成器** - 配置模板基本信息，生成模板代码
3. **属性编辑器** - 添加、删除、修改模板属性
4. **事件编辑器** - 配置模板事件和参数
5. **模板验证** - 验证模板配置是否正确

### 性能监控

1. **性能指标卡片** - 查看关键性能指标和趋势
2. **性能趋势图** - 可视化性能数据变化
3. **实时监控** - 开启/关闭实时性能监控
4. **性能报告** - 生成并下载详细的性能报告

## 🔧 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全的 JavaScript
- **Vite** - 下一代前端构建工具
- **Less** - CSS 预处理器
- **Font Awesome** - 图标库
- **@ldesign/template** - 模板管理系统

## 🎨 自定义样式

应用使用了响应式设计，支持桌面端、平板端和移动端。主要样式变量：

```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  --success-color: #28a745;
  --warning-color: #ffc107;
  --danger-color: #dc3545;
  --info-color: #17a2b8;
}
```

## 📱 响应式支持

- **桌面端** (≥1200px) - 完整功能展示
- **平板端** (768px-1199px) - 适配平板布局
- **移动端** (<768px) - 移动端优化布局
